Exemplo Completo de Tipografia com Tailwind CSS

Este é um exemplo de parágrafo utilizando várias classes de tipografia do Tailwind CSS. Aqui, a fonte é leve, o tamanho é base, a cor é cinza escuro e o texto está justificado. A altura da linha é relaxada e o espaçamento entre letras está normal.

Este parágrafo usa um tamanho de texto maior (extra grande), peso médio, cor azul, alinhamento à direita, altura de linha ajustada e espaçamento entre letras mais amplo.

Este parágrafo apresenta um texto com tamanho grande, peso bold, cor verde, alinhamento à esquerda, transformação em maiúsculas, altura de linha normal, espaçamento entre letras mais apertado e sublinhado.

Por fim, este parágrafo utiliza um tamanho de texto médio, peso normal, cor vermelha, alinhamento centralizado, transformação em minúsculas, altura de linha mais solta, espaçamento entre letras mais amplo e linha através do texto.
